# byt_route

#### 介绍
这是一款信用卡智能还款应用的开源代码，主要功能包括，用户进件还款通道，自行制定还款计划或者输入还款金额智能生成还款计划，每个周期自动跑批还款任务，运行期间达到10万活跃用户，每日跑批百万条还款任务，低硬件成本，高性能运行，欢迎贡献优化
